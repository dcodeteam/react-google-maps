# Pan To

Changes the center of the map to the given `latLng`.

If the change is less than both the width and height of the map, the transition will be smoothly animated.

**Google Maps Docs:**

- [google.maps.Map](https://developers.google.com/maps/documentation/javascript/reference#Map)

  <GoogleMap
    zoom={8}
    maps={google.maps}
    style={{ height: '320px' }}
    center={{ lat: 36.964, lng: -122.015 }}
  >
    <PanTo latLng={{ lat: 36.964, lng: -122.015 }} />
  </GoogleMap>
</Playground>
