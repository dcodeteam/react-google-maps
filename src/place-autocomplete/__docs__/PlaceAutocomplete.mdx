# Place Autocomplete

Attaches place autocomplete to input.

**Google Maps Docs:**

- [google.maps.places.PlaceResult](https://developers.google.com/maps/documentation/javascript/reference/places-service#PlaceResult)
- [google.maps.places.Autocomplete](https://developers.google.com/maps/documentation/javascript/reference/places-widget#Autocomplete)
- [google.maps.places.AutocompleteOptions](https://developers.google.com/maps/documentation/javascript/reference/places-widget#AutocompleteOptions)
- [google.maps.places.ComponentRestrictions](https://developers.google.com/maps/documentation/javascript/reference/places-autocomplete-service#ComponentRestrictions)

<Playground>
  <DocsAPIProvider>
    <PlaceAutocomplete
      placeIdOnly={true}
      strictBounds={true}
      types={["geocode"]}
      componentRestrictions={{ country: "USA" }}
      onPlaceChanged={event => {
        alert(JSON.stringify(event, null, 2));
      }}
      render={({ ref }) => <input ref={ref} type="text" />}
    />
  </DocsAPIProvider>
</Playground>
